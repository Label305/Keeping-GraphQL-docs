# Keeping API
## What is it?
[Keeping](https://keeping.nl) uses [GraphQL](https://graphql.org) to implement an API that allows you to ingegrate Keeping in your own applications through its data. Unlike other APIs that have a set amount of endpoints, Keepings API allows you to compose a query that retrieves exactly the data you want, nothing more, nothing less.
